Como jogar Morra

Como funciona

  • Cada jogador mostra de 0 a 5 dedos simultaneamente
  • Cada jogador aposta o total combinado dos dois
  • O total pode variar de 0 a 10
  • Quem adivinhar o total exato vence a rodada

Faixa de aposta

Como cada jogador mostra de 0 a 5 dedos, o total combinado varia de 0 a 10. Você deve confirmar sua aposta antes de ver a mão do adversário.

Estratégia

Leia as tendências do adversário e varie suas escolhas para ser imprevisível. Se os dois ou nenhum acertar, é empate — nenhum ponto é concedido.
